Summary: Latitude vs. X

This project let u investigate how weather conditions vary as a person gets closer to or approaches the equator. This involved gathering data on more than 500 cities from the OpenWeatherMap API to form a dataset for analysis.

After compiling the dataset, we generated graphs using Matplotlib to depict the variations in weather factors, such as temperature, cloudiness, wind speed, and humidity, concerning latitude. This website provides access to the source data and visualizations generated from our analysis, along with interpretations and descriptions of any patterns and correlations detected.
